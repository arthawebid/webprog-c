## Menggunakan Laravel
1. clone repositories ini
2. copy file .env.example ke .env
```bash
copy .env.example .env
```
4. lakukan install/update modul-modul laravel
```bash
composer install
```
4. lakukan migrate untuk menambahkan sqlite (database)
```bash
php artisan migrate
```
5. generate key
```bash
php artisan key:generate
```
